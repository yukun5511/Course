## 🚀 快速开始

### 环境要求
- Node.js 18+
- Java 17+
- MySQL 8.0+
- Maven 3.6+

### 启动后端

```bash
cd backend
mvn spring-boot:run
```

后端运行在: http://localhost:8081

### 启动前端

```bash
cd frontend
npm install
npm run dev
```

前端运行在: http://localhost:3000
