# 🚀 NATAPP 快速启动（3步）

---

## 第1步：下载 NATAPP

1. 访问：**https://natapp.cn/**
2. 点击顶部 **下载** → **Windows 64位**
3. 解压到：`d:\Myproject\Course\natapp\`

**确认文件**：
```
d:\Myproject\Course\natapp\natapp.exe ✓
```

---

## 第2步：填写 Authtoken

1. 登录 **https://natapp.cn**
2. 进入 **控制台**
3. 复制 **Authtoken**（类似：`a1b2c3d4e5f6g7h8`）
4. 编辑文件：`natapp\config.ini`
5. 替换这行：

```ini
authtoken=您的真实Authtoken
```

**保存文件**

---

## 第3步：启动穿透

**双击运行**：
```
启动NATAPP穿透.bat
```

---

## ✅ 完成！

启动后会显示外网地址：
```
Forwarding: http://abc123.natappfree.cc -> http://localhost:3000
```

**访问管理后台**：
```
http://abc123.natappfree.cc/admin/login
```
