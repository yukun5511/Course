## 方案对比

### ⭐ 方案1：局域网直连（推荐用于同网络测试）

**访问地址**：
```
http://192.168.1.147:3000/admin/login
```

**优势**：
- ✅ 速度最快（局域网，几乎无延迟）
- ✅ 完全稳定
- ✅ 无需任何配置
- ✅ 立即可用

**适用场景**：
- 测试者和您在同一个WiFi/局域网
- 办公室、学校、家庭网络

**登录信息**：
- 账号：`ADMIN001`
- 密码：`admin123`

---

### ⭐⭐ 方案2：使用 Cloudflare Tunnel（免费、快速、稳定）

**Cloudflare Tunnel 优势**：
- ✅ 完全免费
- ✅ 速度快（全球CDN）
- ✅ 稳定可靠
- ✅ 无需域名
- ✅ 支持HTTPS

**设置步骤**：

1. **安装 cloudflared**
   ```powershell
   # 下载
   winget install Cloudflare.cloudflared
   
   # 或者手动下载
   # https://github.com/cloudflare/cloudflared/releases
   ```

2. **快速启动隧道**
   ```powershell
   cloudflared tunnel --url http://localhost:3000
   ```

3. **会生成一个随机的 `.trycloudflare.com` 域名**
   ```
   https://abc123-def456.trycloudflare.com
   ```

4. **分享给测试者即可**

**预计时间**：5分钟

---

### ⭐⭐⭐ 方案3：使用 localtunnel（最简单）

**localtunnel 优势**：
- ✅ 完全免费
- ✅ 无需注册
- ✅ 一行命令启动
- ✅ 速度不错

**设置步骤**：

1. **安装 localtunnel**
   ```powershell
   npm install -g localtunnel
   ```

2. **启动隧道**
   ```powershell
   lt --port 3000
   ```

3. **会生成一个随机域名**
   ```
   https://abc123.loca.lt
   ```

4. **首次访问需要点击确认页面**

**预计时间**：2分钟

---

### ⭐⭐⭐⭐ 方案4：使用 ngrok（国际版）

**ngrok 优势**：
- ✅ 速度快
- ✅ 稳定
- ✅ 免费版可用

**设置步骤**：

1. **注册 ngrok**
   - 访问：https://ngrok.com
   - 免费注册账号

2. **获取 authtoken**
   - 登录后在 Dashboard 找到 authtoken

3. **下载并配置**
   ```powershell
   # 下载
   # https://ngrok.com/download
   
   # 配置 token
   ngrok authtoken YOUR_TOKEN_HERE
   
   # 启动
   ngrok http 3000
   ```

4. **获得外网地址**
   ```
   https://abc123.ngrok-free.app
   ```

**预计时间**：10分钟
