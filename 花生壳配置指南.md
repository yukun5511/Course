# 花生壳外网穿透配置指南

**优势**: 稳定、易用、图形界面、国内速度快

---

## 📥 第1步：下载花生壳

1. 访问：**https://hsk.oray.com/download/**
2. 下载 **Windows 版**
3. 安装（下一步即可）

**或者使用绿色版**：
- 搜索 "花生壳内网穿透 绿色版"
- 下载后直接运行，无需安装

---

## 🔑 第2步：注册并登录

1. 打开花生壳
2. 注册 Oray 账号（手机号或邮箱）
3. 登录账号

---

## ⚙️ 第3步：添加映射

### 方式A：图形界面操作（推荐）

1. 点击 **添加映射**
2. 填写配置：
   - **应用名称**：课程管理系统前端
   - **选择域名**：使用免费域名
   - **内网主机**：`127.0.0.1` 或 `192.168.1.147`
   - **内网端口**：`3000`
   - **外网端口**：选择空闲端口（默认即可）
3. 点击 **确定**

### 方式B：命令行操作

```bash
# 在花生壳安装目录运行
phddns start
phddns add_mapping -name "course-frontend" -proto http -lan_ip 127.0.0.1 -lan_port 3000
```

---

## 🌐 第4步：获取外网地址

添加成功后，花生壳会显示：

```
访问地址: http://your-domain.vicp.cc:端口号
```

**您的外网地址**：
```
http://your-domain.vicp.cc:端口号
```

**管理后台登录**：
```
http://your-domain.vicp.cc:端口号/admin/login
```

## 💡 常见问题

### Q1: 映射添加后无法访问
**解决**：
1. 确认前端服务在3000端口运行
2. 检查防火墙是否允许3000端口
3. 重启花生壳服务

### Q2: 如何停止花生壳
**解决**：
- 图形界面：右键托盘图标 → 退出
- 命令行：`phddns stop`

### Q3: 免费流量用完了怎么办
**解决**：
1. 升级到付费版（¥6/月）
2. 等待下月流量重置
3. 或使用其他工具
