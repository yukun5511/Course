## 🎯 如果还有问题

### 问题1：启动失败

**查看错误日志**，常见原因：
- 端口被占用：修改 `application.yml` 中的端口
- 数据库连接失败：检查 MySQL 是否运行
- 编译错误：运行 `mvn clean compile`

### 问题2：还是500错误

**查看后端窗口日志**，找到具体错误信息

**或者访问 Swagger 文档**：
```
http://localhost:8082/swagger-ui.html
```

### 问题3：ngrok 域名变了

ngrok 每次重启会生成新域名，更新为您当前的域名：
```
https://您的新域名.ngrok-free.dev/api/test/ping
```
