## 🎓 测试最佳实践

### 命名规范
```java
// 格式: test[MethodName]_[Scenario]
@Test
void testCreateActivity_Success() { }

@Test
void testGetActivityById_NotFound() { }
```

### AAA模式
```java
@Test
void testExample() {
    // Arrange - 准备测试数据
    when(mapper.selectById(1L)).thenReturn(entity);
    
    // Act - 执行测试
    Result result = service.getById(1L);
    
    // Assert - 验证结果
    assertNotNull(result);
    assertEquals(1L, result.getId());
}
```

### Mock验证
```java
// 验证方法被调用
verify(mapper, times(1)).insert(any(Entity.class));

// 验证方法未被调用
verify(mapper, never()).deleteById(anyLong());
```
